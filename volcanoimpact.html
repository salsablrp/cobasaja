Sinabung Impacts  


An active volcano located in the Karo Plateau, Karo, North Sumatra.

The eruptions began in 2010 until now and are increasingly active, resulting in a number of buildings, areas, and residents exposed to the volcano impact.

Sinabung Volcano Impacts

The following map was arranged using Mapbox Studio. Data processing was done using the InaSAFE plugin on QGIS by compiling exposure, hazard, and aggregation data. Hazard data used in this part is volcanic hazard map of Sinabung. Exposure data consists of buildings, roads, land cover, and population data. Aggregation data consists of administrative data combined with gender ratio data.
